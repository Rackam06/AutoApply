# docs/

Place your CV PDFs here (they are gitignored — never commit personal CVs).

Expected defaults (overridable in `.env` or the dashboard):

| File | Used when |
|---|---|
| `CV_English.pdf` | Lead country is not France |
| `CV_French.pdf` | Lead country is France |

You can also upload CVs from the Streamlit **Your Profile** section; uploads are saved into this folder.
